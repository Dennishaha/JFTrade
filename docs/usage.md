# 使用指南

JFTrade 控制台围绕五类常用任务展开：连接券商、看行情、管理策略、执行回测，以及查看系统状态。

## 首次使用建议顺序

1. 打开控制台并完成管理员登录。
2. 到系统或设置页面确认 sidecar 和 OpenD 状态正常。
3. 配置 Futu / OpenD 集成，再进入行情、策略或回测功能。

## 功能入口

### 账户与交易

- 查看券商连接状态、账户、持仓和订单。
- 核对交易账户是否已经和当前 broker 配置正确绑定。

### 行情

- 查看快照、K 线、盘口和实时订阅状态。
- 观察实时连接是否正常、行情是否在持续刷新。

### 策略

- 编辑策略定义。
- 绑定标的、账户和运行参数。
- 管理策略实例的启动、暂停、停止和删除。

### 回测

- 启动回测任务。
- 查看运行进度、结果和历史记录。

### 系统

- 查看运行时状态、OpenD 健康信息和排障入口。
- 在需要时进入文档、Swagger 或诊断页面。

# 使用指南

JFTrade 控制台围绕六类常用任务展开：管理自选、连接券商、看行情、管理策略、执行回测，以及查看系统状态。

## 首次使用建议顺序

1. 打开 JFTrade 桌面应用；桌面模式无需登录。
2. 到系统或设置页面确认 sidecar 和 OpenD 状态正常。
3. 配置 Futu / OpenD 集成，再进入行情、策略或回测功能。

只有需要从浏览器访问时，才到“设置 → Web 访问”确认对外端口、设置密码并主动开启。保存后监听器立即生效，浏览器使用该 Web 访问密码登录，桌面应用仍然无需密码。默认地址是 `http://127.0.0.1:6688`；Web 关闭时该桌面浏览器端口不会监听。

## 功能入口

### 自选

- 在 `/watchlist` 管理跨市场、多分组自选，搜索标的并查看可见行快照行情。
- 在工作台左侧自选栏快速切换标的；点击图表标题旁的星标编辑当前标的所属分组。
- 从 Futu 读取远端分组后先预览差异，再确认导入；默认不会删除本地独有项，也不会写回券商。

完整语义见 [自选系统](./watchlist.md)。

### 账户与交易

- 查看券商连接状态、账户、持仓和订单。
- 核对交易账户是否已经和当前 broker 配置正确绑定。

### 行情

- 查看快照、K 线、盘口和实时订阅状态。
- 观察实时连接是否正常、行情是否在持续刷新。

### 策略

- 编辑策略定义。
- 绑定标的、账户和运行参数。
- 管理策略实例的启动、暂停、停止和删除。

### 回测

- 启动回测任务。
- 查看运行进度、结果和历史记录。

### 系统

- 查看运行时状态、OpenD 健康信息和排障入口。
- 在需要时进入文档、Swagger 或诊断页面。
